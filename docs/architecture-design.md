## 1. жҰӮиҝ°

### 1.1 йЎ№зӣ®иғҢжҷҜ

Edgion жҳҜдёҖдёӘеҹәдәҺ Rust е’Ң Pingora зҡ„й«ҳжҖ§иғҪ API зҪ‘е…іпјҢе®һзҺ°дәҶе®Ңж•ҙзҡ„ Kubernetes Gateway API ж ҮеҮҶгҖӮдёәдәҶжҸҗеҚҮиҝҗз»ҙж•ҲзҺҮе’Ңз”ЁжҲ·дҪ“йӘҢпјҢйңҖиҰҒдёә `edgion-controller` ж·»еҠ  Web з®ЎзҗҶз•ҢйқўпјҢж”ҜжҢҒеҸҜи§ҶеҢ–зҡ„иө„жәҗз®ЎзҗҶгҖӮ

### 1.2 и®ҫи®Ўзӣ®ж Ү

- **жҳ“з”ЁжҖ§**пјҡзӣҙи§Ӯзҡ„з”ЁжҲ·з•ҢйқўпјҢйҷҚдҪҺиҝҗз»ҙй—Ёж§ӣ
- **е®Ңж•ҙжҖ§**пјҡж”ҜжҢҒжүҖжңү Gateway API иө„жәҗзҡ„ CRUD ж“ҚдҪң
- **е®үе…ЁжҖ§**пјҡж•°жҚ®йӘҢиҜҒгҖҒй”ҷиҜҜеӨ„зҗҶгҖҒпјҲеҗҺз»ӯпјүжқғйҷҗжҺ§еҲ¶
- **жҖ§иғҪ**пјҡеүҚз«ҜдјҳеҢ–гҖҒеҗҺз«Ҝй«ҳж•ҲгҖҒеӨ§йҮҸиө„жәҗеңәжҷҜдёӢжөҒз•…
- **йғЁзҪІз®Җдҫҝ**пјҡеөҢе…ҘејҸйғЁзҪІпјҢеҚ•дёҖдәҢиҝӣеҲ¶ж–Үд»¶

### 1.3 еҸӮиҖғйЎ№зӣ®

еҹәдәҺдёҡз•Ңдё»жөҒ API зҪ‘е…ізҡ„е®һи·өз»ҸйӘҢпјҡ

```rust
use rust_embed::RustEmbed;

#[derive(RustEmbed)]
#[folder = "../edgion-dashboard/dist"]
struct Assets;

async fn serve_static(path: &str) -> Response {
    match Assets::get(path) {
        Some(content) => {
            let mime = mime_guess::from_path(path).first_or_octet_stream();
            Response::builder()
                .header(header::CONTENT_TYPE, mime.as_ref())
                .body(Body::from(content.data))
                .unwrap()
        }
        None => Response::builder()
            .status(StatusCode::NOT_FOUND)
            .body(Body::from("404 Not Found"))
            .unwrap(),
    }
}
```
